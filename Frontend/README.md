# 🎨 Frontend - Grocery App

This is the frontend of the Grocery E-Commerce application built using React and Tailwind CSS.

---

## 🚀 Features

- Product listing
- Search functionality
- Category filtering
- Pagination
- Add to cart
- Cart management
- Responsive UI

---

## 🧱 Tech Stack

- React.js
- Tailwind CSS
- React Router
- React Hot Toast

---

## ⚙️ Setup

```bash
npm install
npm run dev
```
--- 

## 📁 Folder Structure
- src/
- ├── components/
- ├── pages/
- ├── utils/
- └── App.jsx

--- 
## 🔗 API

Make sure backend is running on:

```bash
http://localhost:3000
```
--- 

## ✨ Notes
- Uses JWT token from localStorage
- Search and filter handled on frontend
